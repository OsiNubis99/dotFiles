## Getting Started

### Prerequisites

You need any Fedora Linux distribution.

### Deployment

1. Clone this repository

```sh
git clone https://github.com/OsiNubis99/dotFiles.git ~/dotFiles
```

2. Run the installer

```sh
bash ~/dotFiles/install.sh
```
